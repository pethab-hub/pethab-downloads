# PetHab Downloads

PetHab package delivery has moved to note articles.

This public repository no longer hosts character ZIP files. Free starter PETS
are attached directly to the public note article, and member-only packages are
delivered through note membership or another restricted channel.

## Current Delivery

- Free starter ZIPs: attached in the public PetHab note article
- Member benefits: note member-only posts or restricted delivery
- Internal package archive: private GitHub repository

## Public Scope

Do not publish the full character library or member-benefit ZIPs here.

The public free starter set remains limited to the approved starter PETS:

- White Sprint Runner
- AI Ops Foxcat
- Battery Dock Nurse
- Trackball Homebody
- Mail Courier Muse

Some free starter ZIPs include PC background images. Those ZIPs are now handled
as note attachments, not public GitHub downloads.

## Use

Open the PetHab note article, download the attached ZIP for the PET you want to
try, extract it locally, and follow the included README.

Redistribution, resale as-is, or claims of official endorsement are not included.
