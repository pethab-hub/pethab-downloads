# PetHab Downloads

Public download repository for PetHab character packages.

## Download

Download the `download.zip` file for the character package you want to use.
Each package now has a stable package number such as `PETHAB-DL-001` so X/social posts, gallery images, and model rollout notes can point back to the exact download package.

## Use

1. Download a character ZIP.
2. Extract the ZIP locally.
3. Add the extracted character folder to your local PETS folder.
4. Restart or refresh the host app and select the character.

If the package includes PC backgrounds, open `bonus/README-ja.md` or `wallpapers/README.md` in the extracted folder and manually select the background. Wallpaper setup is not automatic.
